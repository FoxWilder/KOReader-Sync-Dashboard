Installation

Run this command in PowerShell to install or upgrade:

Why this fork?

  • Native Performance: Direct runtime execution on Windows Server.
  • Automated Lifecycle: Built-in GitHub Action pipelines for releases and previews.
  • Zero Config: Switches to local SQLite and filesystem storage automatically.
